# Plan: clean command for envoy-mini-builder

Add a `clean` subcommand that SSHes to the mini and runs `bazel clean --expunge`
on all envoy builder workspaces. Wired as a scheduled GH Actions workflow in
dio/envoy-builder that runs twice a week, 4 hours before the darwin nightly build.

## What it does

SSH to the mini, find all Bazel workspaces under `~/envoy-builder/`, run
`bazel clean --expunge` in each. This wipes the local action cache and output
base. BuildBuddy remote cache re-warms on the next build.

## Files to change

### cmd/clean.go (new)

```go
package cmd

import (
    "fmt"
    "github.com/dio/envoy-mini-builder/internal/mini"
    "github.com/spf13/cobra"
)

type cleanFlags struct {
    sshHost string
    sshPort int
}

var cf cleanFlags

var cleanCmd = &cobra.Command{
    Use:   "clean",
    Short: "Clean Bazel cache on the remote host",
    RunE:  runClean,
}

func init() {
    f := cleanCmd.Flags()
    f.StringVar(&cf.sshHost, "host", "dio@mini", "SSH host")
    f.IntVar(&cf.sshPort, "port", 22, "SSH port")
    rootCmd.AddCommand(cleanCmd)
}

func runClean(cmd *cobra.Command, args []string) error {
    header("Clean Bazel cache on %s", cf.sshHost)
    c := mini.NewCleaner(mini.CleanConfig{
        SSHHost: cf.sshHost,
        SSHPort: cf.sshPort,
    })
    return c.Run(cmd.Context())
}
```

### internal/mini/cleaner.go (new)

CleanConfig mirrors the SSH fields from Config.

Cleaner.Run SSHes and runs:

```sh
set -euo pipefail
shopt -s nullglob
for src in ~/envoy-builder/*/src; do
  if [[ -f "$src/WORKSPACE" || -f "$src/MODULE.bazel" ]]; then
    echo "→ cleaning $src"
    cd "$src"
    bazel clean --expunge 2>&1 || echo "  (bazel not found or failed, skipping)"
  fi
done
echo "→ done"
```

Uses the same sshArgs / execScript pattern as Builder — pipe script to bash,
stream stdout live, no sentinel needed (exit code is sufficient).

## New workflow: .github/workflows/clean-cache.yml in dio/envoy-builder

```yaml
name: Clean Bazel Cache

on:
  schedule:
    - cron: '0 22 * * 0,3'  # Sunday and Wednesday 22:00 UTC
  workflow_dispatch:

jobs:
  clean:
    runs-on: ubuntu-latest
    timeout-minutes: 30
    steps:
      - uses: tailscale/github-action@... # v3
        with:
          oauth-client-id: ${{ secrets.TS_OAUTH_CLIENT_ID }}
          oauth-secret: ${{ secrets.TS_OAUTH_SECRET }}
          tags: tag:ci
      - name: Set up SSH key
        run: |
          mkdir -p ~/.ssh
          printf '%s\n' "${{ secrets.MINI_SSH_KEY }}" > ~/.ssh/id_ed25519
          chmod 600 ~/.ssh/id_ed25519
      - name: Install envoy-mini-builder
        env:
          GH_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: |
          gh release download \
            --repo dio/envoy-mini-builder \
            --pattern '*linux_amd64*.tar.gz' \
            --output /tmp/emb.tar.gz
          tar -xzf /tmp/emb.tar.gz -C /tmp envoy-mini-builder
          sudo mv /tmp/envoy-mini-builder /usr/local/bin/envoy-mini-builder
      - name: Clean
        run: envoy-mini-builder clean --host dio@mini
```

## Step-by-step

3. go build + go test.

6. Add .github/workflows/clean-cache.yml to dio/envoy-builder.
